# Distilled — Style Extractor

Reads a body of the user's writing and produces two outputs: a compact 64-dimensional style vector (for downstream filters and DPO rejection-paraphrasing) and a short human-readable style card (shown in Kiln's Style Profile panel).

See `.claude/skills/distillation-pipeline/` for the operational recipe.

## Input / output

Input: UTF-8 text, 300–4,000 characters (longer samples are chunked and averaged).

Output:

```json
{
  "vector": [0.12, -0.08, ..., 0.31],
  "card": {
    "summary": "You write in short, declarative sentences. You use semicolons twice as often as average. You hedge rarely.",
    "traits": [
      {"trait": "sentence_length", "value": "short", "evidence": "Median sentence length 11 words."},
      {"trait": "formality", "value": "neutral", "evidence": "Contractions present; second-person address rare."},
      {"trait": "hedging", "value": "rare", "evidence": "No 'I think' or 'perhaps' in 120 sentences."}
    ]
  }
}
```

- `vector[64]` — floats in `[-1, 1]`. Used downstream.
- `card` — rendered in SwiftUI. Three traits is typical; up to five supported.

## Architecture (M9.C shipped path)

- Embedder: `sentence-transformers/all-MiniLM-L6-v2` (384-dim).
- Head: scikit-learn `MultiOutputRegressor(Ridge)` over six style axes — `formality`, `verbosity`, `warmth`, `hedging`, `humor`, `directness` — each in `[0, 1]`.
- Shipped as `packages/kiln_trainer/artifacts/style-regressor.pkl`. The original spec also envisioned a Qwen LoRA card head; that path is post-M9 work — for now the human-readable card is composed deterministically from the descriptors via `StyleSignaturePresenter`.

## Training data origin

- 1,500 writing samples generated by `scripts/opus-distill/build_style_input.py` (six style categories × parametric templates × seeded fills).
- Opus 4.7 produces the descriptor vector per sample.
- Split: 80/20. Held-out test set frozen at label-creation time.

## Artifacts

- `packages/kiln_trainer/artifacts/style-regressor.pkl` — shipped.
- `manifest.json` — version, git SHA, Opus teacher id, training counts, metrics, ship-bar status. Generated by `scripts/build_distilled_manifests.py`.
- `raw_labels.jsonl` — gitignored; produced by the style-extractor Managed Agent.

## Runtime use

Runs once when the user completes ingest. Output feeds the Style Profile card UI (via `StyleSignaturePresenter`).
